# Getting-and-Cleaning-Data-Assigment

## Project Description and Introduction

This project and all its various files are made available which indicates the type of work performed, and details of the analysis. The raw data was taken as described in the CodeBook.Rmd file. Basically, this analysis is as done as what was instructed in the Getting and Cleaning Data Course Project and includes all of the various files. 
The "run_analysis.R" file is the actual code which does the analysis.
The "CodeBook.Rmd" file is the codebook file explaining the various steps in the analysis, and information on the variables used in the tidy dataset.
The "codebook variables info.csv" file is the file from which the codebook dataframe table is generated. It includes the variable names, class, units, and description of all of the variables.
